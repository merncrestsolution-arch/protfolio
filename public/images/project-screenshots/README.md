## Specifications

- **Format**: JPG or PNG
- **Dimensions**: 1200x675px (16:9 aspect ratio)
- **File Size**: Under 500KB per image
- **Content**: Full application screenshot or hero section

## Tips

1. Capture clean, professional screenshots
2. Include key UI elements and features
3. Use consistent styling across all screenshots
4. Optimize images with tools like TinyPNG or ImageOptim
5. Ensure text is readable in the screenshot

If a screenshot is missing, a placeholder will be displayed automatically.
